# 使い方
* exe
  * ダブルクリックで実行するとユーザIDとパスワードを入力を求められるので入力
* bat
  * batを編集し、[userid] [password]を自分のものに書き換えて実行

# 実行結果
* ranking.csvが作成されます。
  * 内容の確認にはExcelやGoogleスプレッドを使ってください。

## 内容
* 各曲各難易度毎のランキングと1位との差が出力されます
  * 例)
    * 315位で1位との点差が1224 ⇒ 315(-1224)
    * 1位で1位との点差が0 ⇒ 1(0)
    * 未プレイ ⇒ -(1位との差)
